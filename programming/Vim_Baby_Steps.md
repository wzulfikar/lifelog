# Vim Baby Steps
- start with `:` and then command. eg:
  - to edit, type in `:e` and then filename
- store configuration in your `~/.vimrc` file

vim-vinegar (must-have): 
- `-` enter vim-vinegar (display current directory)
- `⬆ + d` : delete file/directory
- `d` : create directory
- `⬆ + 5` : create directory
